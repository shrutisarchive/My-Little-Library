  My-Little-Library
 Little free library
Your Cozy Little Corner

This is a simple file-based library management system built in C using structured data and modular functions and it can be extended using data structures for optimization

## Features of MY Little Library
- Membership
- Books issue
- Books return
- View your record
- A clean console UI

## Concepts Used
- File Handling in C
- Structures
- Functions

## Folders
My Little Library:
- include/
- source/
- data/ (ignored in git)
- README.md

## How to Run

```bash
cd source
gcc main.c functions.c -o run
./run
```
